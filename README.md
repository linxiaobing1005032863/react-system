# 后台管理系统安装和启动方式
# 环境依赖:
```
安装 nodejs v6.12.3版本
安装 yarn
```
# 项目初始化方式:
```
安装依赖包： yarn
```
# 开发模式运行:
```
yarn run dev
```
# 线上打包:
Mac / Linux系统：
```
yarn run dist
```
Windows系统：
```
yarn run dist_win
```
# 技术栈：
```
react16 + react-router4 + redux + webpack + yarn + sass
```
#架构:
    采用了前后端分离，分层化设计，模块化开发
