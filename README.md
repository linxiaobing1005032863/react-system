# 后台管理系统安装和启动方式
## 环境依赖:
```
安装 nodejs v6.12.3版本
安装 yarn
```
## 项目初始化方式:
```
安装依赖包： yarn
```
## 开发模式运行:
```
yarn run dev
```
## 线上打包:
Mac / Linux系统：
```
yarn run dist
```
Windows系统：
```
yarn run dist_win
```
# 技术及主要功能
## 技术栈：
```
react16 + react-router4 + redux + webpack + yarn + sass
```
## 架构:
    采用了前后端分离，分层化设计，模块化开发
## 主要功能模块
```
    1.用户模块（登录，退出，用户列表）
    2.商品模块（商品列表，商品详情，添加/修改商品）
    3.品类模块（品类列表，添加品类，修改品类名称）
    4.订单模块（订单列表，订单详情，发货）
```
